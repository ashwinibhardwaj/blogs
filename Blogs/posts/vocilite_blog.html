Vocilite: Simple Text Explanations with Voice

Posted on: July 13, 2025

My Story Behind Vocilite

Spent this weekend learning and building something new! I was reading a technical article full of jargon and wished there was an easier way to understand things without breaking my reading flow. That sparked the idea for Vocilite.

Vocilite is a Chrome extension (still in development) that lets you highlight any text on a webpage and instantly get a simple explanation—and even have it read out loud. It's like having a personal tutor on any website, helping you stay focused and learn faster.

Vocilite isn’t live yet, but I’ve been making solid progress. You can explore the code or follow along with the journey on GitHub: Vocilite GitHub Repository.

The Problem I Wanted to Solve

So much of the internet is packed with complicated language—research papers, legal documents, technical blogs—that can slow you down or leave you feeling lost. I wanted to make things simpler by solving a few key problems:

  • Helping users understand tricky terms or sentences instantly, without leaving the page.
  • Providing plain-English explanations instead of dictionary definitions.
  • Offering spoken explanations for better accessibility and multitasking.

My goal was to help students, professionals, or anyone stay in their reading flow while learning new things effortlessly.

My Solution: Vocilite

Vocilite is a Chrome extension designed to help you highlight any text and get a simple, conversational explanation right there on the page. It also reads it aloud in clear, natural voices—including Indian English accents for better local clarity.

  • Highlight and Explain: Select any text and click “Explain.”
  • Clear Explanations: The AI generates short, jargon-free explanations you can easily understand.
  • Text-to-Speech: Hear the explanation read out loud for better comprehension or multitasking.
  • Powered by AI: Built using LangChain, Groq’s LLaMA 3, and a Flask backend for quick responses.
  • Clean Audio Output: Vocilite cleans up AI text so your voice assistant doesn’t read out extra symbols or markdown.

How I Built Vocilite

I wanted to learn some new tech skills while solving a problem that I personally face. So I built Vocilite in a few simple steps:

  • I started with a basic Chrome extension that detects highlighted text on any webpage.
  • When you click “Explain,” the extension sends the text to my Flask backend.
  • My backend uses LangChain and Groq’s LLaMA 3 to generate a simple explanation.
  • The cleaned explanation is sent back to the extension and displayed to the user.
  • I also used the Web Speech API to let Vocilite read the explanation out loud.

It’s been fun experimenting with how AI and voice technology can make learning smoother and more accessible!

How It Works

  1. Step 1: Install the Vocilite Chrome extension (coming soon!).
  2. Step 2: Highlight any text you’d like explained.
  3. Step 3: Click the “Explain” button in Vocilite’s floating panel.
  4. Step 4: See a simple explanation—or listen to it spoken aloud instantly.

Challenges I Faced

  • Cleaning AI-generated text so that symbols or formatting marks aren’t read aloud awkwardly.
  • Finding high-quality voices for different accents, especially Indian English.
  • Keeping explanations short but still informative and clear.
  • Designing a lightweight UI that doesn’t feel intrusive while browsing.

What I Learned

  • Integrating LangChain and Groq’s LLaMA 3 for practical, real-time apps.
  • Building Chrome extensions and handling complex DOM interactions smoothly.
  • Cleaning and refining AI text for a better text-to-speech experience.
  • Creating tools that make technical content more accessible for everyone.
